Introduction

Upzet is a fully featured premium admin dashboard template in Django with developer-friendly codes.

Upzet is an admin dashboard template that is a beautifully crafted, clean & minimal designed admin template with Dark, Light Layouts with RTL supported. The theme includes beautiful dashboard, many ready to use hand crafted components, UI elements, widgets, pages, etc.

It will help your team moving faster and saving development costs and valuable time. If you’re a developer and looking for an admin dashboard that is fully responsive with Bootstrap ^5.1.3 and Django ^3.2.9 & Python ^3.8.10 then you are at the right place to start your project using Upzet - Django Responsive Bootstrap Admin Dashboard.
