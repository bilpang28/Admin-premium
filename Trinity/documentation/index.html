HTML Structure

Trinity is built on it's own bootstrap grid system. The syntax is simple and it's effective cross browser. You can easily customize all parts of your site or build a new ones.

1.1 Basic HTML template

<!DOCTYPE html>
<html lang="en">
    <head>
        <title>Trinity</title>
        <meta charset="utf-8">
        <meta name="description" content="Trinity">
        <meta http-equiv="X-UA-Compatible" content="IE=edge">
        <meta name="viewport" content="width=device-width, initial-scale=1">
        <!-- favicon-->
        <link rel="shortcut icon" href="favicon/favicon.ico">
        <!-- ./ favicon-->
        <link rel="stylesheet" href="css/bootstrap.css">
        <link rel="stylesheet" href="css/main.css">
    </head>
    <body>
        <div class="all">
            <!-- header-->
            <div class="header"><!-- Navigation --></div>
            <!-- ./ header-->
            <!-- search-->
            <div id="search" class="search-form">...</div>
            <!-- ./ search-->
            <!-- content-->
            <div class="page">
                <div class="page-nav"><!-- Page navigation --></div>
                <div class="page-header"><!-- Page title --></div>
                <div class="page-content"><!-- Page content --></div>
            </div>
            <!-- ./ content-->
            <!-- footer-->
            <div class="footer">...</div>
            <!-- ./footer-->
            <!-- ./ all content-->
        </div>
        <!-- scripts-->
        <script src="js/jquery.min.js"></script>
        <script src="js/bootstrap.min.js"></script>
        <script src="js/slick.min.js"></script>
        <script src="js/main.min.js"></script>
        <!-- ./ scripts-->
    </body>
</html>

Font Usage

Fonts used in Trinity HTML template:

  • Roboto — for all text

Check it on the Google Fonts.

These fonts are embedded into style.css with this code:

@import url('https://fonts.googleapis.com/css?family=Roboto:100,100i,300,300i,400,400i,500,500i,700,700i,900,900i&subset=cyrillic,cyrillic-ext,greek,greek-ext,latin-ext,vietnamese');
